    1. Home
    2. Features
    3. Syntax Highlighting

    metalsmith-metallic will highlight code in markdown files. The color scheme can be selected from any highlight.js theme, and set in style/vendors. Supported languages can be found in the highlight.js documentation.

    Markup can include the name of, or an alias of, a language with or without a space in front of it. For examples, see the source of this page.
